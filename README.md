## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/) (recommended)
- A modern browser (Chrome, Firefox, Edge, etc.)

### Installation

1. Clone this repository:

    ```bash
    git clone https://github.com/DisanduP/3js-Solider.git
    cd 3js-Solider
    ```

2. Install dependencies (if any):

    ```bash
    npm install
    ```

3. Start a local server:

    ```bash
    npm start
    ```

4. Open your browser and visit `http://localhost:3000` or as otherwise specified.

### If not using Node.js

1. Simply open `index.html` in your browser if this is a plain HTML/JavaScript project.
